En este portafolio se muestran algunos de mis trabajos realizados durante mis 3 años en la beca ¡Supérate! Raíces.

Trabajos

Durante mis tres años trabajé en diferentes proyectos, por ejemplo en Scratch, Filmora, Adobe illustrator, que son dirigidos al Diseño, y de igual manera realicé proyectos de creación de páginas web y programación en diferentes aplicaciones como Sublime y Atom en las cuales demostré mis habilidades en manejar código html, css y un poco de javascript.

Filmora

Con la aplicación de Filmora aprendimos a hacer edición de videos, así como usar la pantalla verde, usar transiciones, recortar e importar audio a los videos.

Adobe Illustrator

En Illustrator aprendimos a crear logos y a usar las diferentes herramientas de la aplicación, así como usar las capas para crear las imágenes necesitadas.

Sublime Text

Sublime text es una herramienta para crear código html el cuál nos sería útil durante nuestros 2 años restantes para la creación de páginas web.

Excel

Excel además de ser una de las primeras certificaciones que tuvimos de igual manera la aprendimos para crear y manejar tablas de información, como editarlas y crear gráficos.
